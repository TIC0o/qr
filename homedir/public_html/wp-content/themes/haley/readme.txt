== Description ==

Haley is an elegant and minimal WordPress theme, perfect for creating personal, fashion, travel, or professional blog websites. It offers an intuitive user interface with numerous customization options through the WordPress Live Customizer. Compatible with WooCommerce and fully optimized for mobile devices, Haley is the ideal choice for those looking for a modern and functional website.
